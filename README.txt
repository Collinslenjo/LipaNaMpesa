== Description ==

This is WooCommerce payment gateway enabling payments of the Mpesa buy goods via the Kopokopo api.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `kopo_plugin` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== features ==

features:

1. Pay via Lipa na MPESA
2. uses WooCommerce
3. works with the Kopokopo Api
